## Projects

### [news-intelligence](https://github.com/nborders/news-intelligence)
Wikipedia-anchored daily news analysis workflow. Three Python scripts that fetch Wikipedia's Current Events portal, bundle multi-source news (NPR, Meduza, Global Times, ISW, Telegram), and call Claude to produce a footnoted briefing. Renders as a dark-mode browser page with clickable sources.

### finance-monitor *(private)*
Local net worth dashboard. Flask + SQLite + WebSocket live updates. Pulls from Coinbase API, Plaid-connected bank/brokerage accounts, and Fidelity CSV exports. Charts net worth history, asset allocation, and unrealized P&L across crypto, brokerage, and retirement accounts.

### [PacificMount](https://github.com/nborders/Pacific-EQ-Mount)
DIY harmonic-drive equatorial telescope mount — my own design, inspired by the AlkaidMount. Built for astrophotography from Portland.
